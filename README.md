# Flutter TO DO
Este é um aplicativo simples de lista de tarefas desenvolvido em Flutter. É o meu primeiro aplicativo em Flutter e utiliza as bibliotecas Slidable e Hive.

## Funcionalidades

O aplicativo possui as seguintes funcionalidades:

 - Adicionar tarefa: Você pode adicionar uma nova tarefa à lista. Basta digitar o nome da tarefa e pressionar o botão "Adicionar". A tarefa será salva automaticamente e exibida na lista.

 - Excluir tarefa: Se você deseja remover uma tarefa da lista, basta deslizar o item para a esquerda e tocar no ícone de lixeira. A tarefa será excluída permanentemente.

 - Marcar tarefa como concluída: Quando você concluir uma tarefa, basta cliar na checkbox da tarefa. A tarefa será marcada como concluída e será exibida visualmente na lista.

## Dependências

 - [flutter_slidable ](https://pub.dev/packages/flutter_slidable) : Biblioteca para adicionar a funcionalidade de deslizar os itens da lista.

 - [hive](https://pub.dev/packages/hive) e [hive_flutter](https://pub.dev/packages/hive_flutter): Biblioteca para persistência de dados localmente. Utilizada para armazenar as tarefas do aplicativo.

## Como executar o projeto localmente

Para executar o aplicativo em seu dispositivo local, siga estas etapas:

 1. Certifique-se de ter o Flutter instalado em seu sistema. Caso contrário, siga as instruções na documentação oficial do Flutter.

 2. Clone este repositório em seu ambiente local:
``` 
git clone https://github.com/WiliamMelo01/todo.git
```

3. Navegue até o diretório do projeto:
``` 
cd todo
```

4. Execute o seguinte comando para baixar as dependências do projeto:
``` 
flutter pub get
```

5. Conecte seu dispositivo ou inicie um emulador.

6. Execute o aplicativo usando o seguinte comando:
``` 
flutter run
```
